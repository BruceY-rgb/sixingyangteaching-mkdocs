指针基础概念练习题

一、选择题（每题5分，共20分）

  1. 以下关于指针的描述，正确的是？
    A. 指针是存储变量地址的变量
    B. 指针可以直接存储整数值
    C. 指针的大小与它所指向的数据类型无关
    D. 指针变量名前面需要加*声明

答案：A
解析：指针存储内存地址，声明时用*，但变量名本身不加*。指针大小通常固定（如4/8字节），与类型无关，但选项C表述不严谨（应为"sizeof(指针)"与类型无关）。

  1. 若有int a = 10; int *p = &a;，则*p的值是？
    A. a的地址
    B. 10
    C. p的地址
    D. 随机值

答案：B
解析：*p表示解引用，获取p指向的值（即a的值10）。

  1. 以下代码的输出是？
    c int x = 5, y = 10; int *ptr = &x; *ptr = y; printf("%d", x);
    A. 5
    B. 10
    C. x的地址
    D. 编译错误

答案：B
解析：ptr指向x，*ptr = y修改了x的值为10。

  1. 关于NULL指针，错误的是？
    A. NULL指针不指向任何有效内存
    B. 解引用NULL指针会导致运行时错误
    C. NULL与0等价
    D. malloc()失败时返回NULL

答案：C
解析：NULL在C中通常是(void*)0，与整数0不等价（但比较时行为相同）。

二、填空题（每空5分，共30分）

  1. 补全代码，通过指针交换两个变量的值：
    c void swap(int *a, int *b) { int temp = ______; // 填空1 ______ = *b; // 填空2 *b = temp; }
    答案：
  2. 填空1：*a
  3. 填空2：*a
  1. 补全指针声明：
    c int x = 100; ______ ptr = &x; // 声明指向x的指针 printf("%d", ______); // 输出x的值
    答案：
  2. 第一空：int *
  3. 第二空：*ptr
  1. 以下代码的输出是：
    c char str[] = "Hello"; char *p = str; printf("%c", *(p + 2));
    答案：l
    解析：p + 2指向第3个字符'l'。

三、编程题（50分）

  1. 指针基础操作（20分）
  2. 声明一个整数变量num并赋值为50
  3. 声明指针p指向num
  4. 通过指针修改num的值为100
  5. 输出num的值和地址

参考代码：
c #include <stdio.h> int main() { int num = 50; int *p = &num;//定义一个指针变量，指针指向num *p = 100;//对p指向的变量赋值 printf("值: %d, 地址: %p", num, &num); return 0; }

  1. 指针与数组（30分）
  2. 定义一个包含{3, 7, 2, 9}的整型数组
  3. 使用指针遍历数组并计算元素总和
  4. 输出总和及最大值的地址

参考代码：
```c #include int main() { int arr[] = {3, 7, 2, 9}; int ptr = arr, sum = 0, max = ptr; int *max_ptr = ptr;

   for (int i = 0; i < 4; i++) {
       sum += *(ptr + i);//*(ptr+i)<=>arr[i]
       if (*(ptr + i) > max) {
           max = *(ptr + i);
           max_ptr = ptr + i;
       }
   }

   printf("总和: %d\n最大值地址: %p", sum, (void*)max_ptr);
   return 0;

} ```

答案及解析

选择题解析

  1. A：指针的本质是存储地址的变量。
  2. B：*p解引用获取值。
  3. B：通过指针修改了x的值。
  4. C：NULL是空指针常量，与整数0类型不同但值相同。

编程题要点

  • 指针声明：类型*变量名（如int *p）
  • 解引用：用*访问指针指向的值
  • 指针运算：ptr + i等价于&arr[i]
  • 地址输出：使用%p和(void*)强制转换
