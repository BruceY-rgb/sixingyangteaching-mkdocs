作业10

一、选择题（每题5分，共20分）

  1. 以下关于指针的描述，正确的是？
    A. 指针是存储变量地址的变量
    B. 指针可以直接存储整数值
    C. 指针的大小与它所指向的数据类型无关
    D. 指针变量名前面需要加*声明
  1. 若有int a = 10; int *p = &a;，则*p的值是？
    A. a的地址
    B. 10
    C. p的地址
    D. 随机值
  1. 以下代码的输出是？
    c int x = 5, y = 10; int *ptr = &x; *ptr = y; printf("%d", x);
    A. 5
    B. 10
    C. x的地址
    D. 编译错误
  1. 关于NULL指针，错误的是？
    A. NULL指针不指向任何有效内存
    B. 解引用NULL指针会导致运行时错误
    C. NULL与0等价
    D. malloc()失败时返回NULL

二、填空题（每空5分，共30分）

  1. 补全代码，通过指针交换两个变量的值：
    c void swap(int *a, int *b) { int temp = ______; // 填空1 ______ = *b; // 填空2 *b = temp; }
  1. 补全指针声明：
    c int x = 100; ______ ptr = &x; // 声明指向x的指针 printf("%d", ______); // 输出x的值
  1. 以下代码的输出是：
    c char str[] = "Hello"; char *p = str; printf("%c", *(p + 2));

三、编程题（50分）

  1. 指针基础操作（20分）
  2. 声明一个整数变量num并赋值为50
  3. 声明指针p指向num
  4. 通过指针修改num的值为100

  5. 输出num的值和地址

  6. 指针与数组（30分）

  7. 定义一个包含{3, 7, 2, 9}的整型数组
  8. 使用指针遍历数组并计算元素总和
  9. 输出总和及最大值的地址
