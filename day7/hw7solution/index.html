一、选择题（每题5分，共20分）

  1. 关于全局变量和局部变量，以下说法正确的是？
    A. 全局变量未初始化时值为随机数
    B. 局部变量的作用域仅限于定义它的函数
    C. 全局变量不能与局部变量同名
    D. 局部变量的生命周期从程序开始到结束

答案：B
解析：
- A错误：全局变量未初始化时默认为0。
- C错误：可以同名，局部变量会屏蔽全局变量。
- D错误：局部变量生命周期仅在函数执行期间。

  1. 以下代码的输出是？
    c int a = 5; void func() { int a = 10; printf("%d", a++); } int main() { func(); printf("%d", a); return 0; }
    A. 10 5
    B. 11 5
    C. 10 6
    D. 11 6

答案：A
解析：
- func() 中 a++ 先输出局部变量 10，再自增为11（但未使用）。
- main() 输出全局变量 a（仍为5）。

  1. i++ 和 ++i 的区别是？
    A. i++ 返回自增前的值，++i 返回自增后的值
    B. i++ 比 ++i 执行速度快
    C. 两者完全等价
    D. i++ 只能用于循环语句

答案：A
