## 🎯 今日精选
```c
# include <stdio.h>
int main (void)
{
    int sum = 0;
    for(int i = 0;i < 100;i++)
    {
        sum = sum + i;
    }
    
    printf("1+2+...+100=%d",sum); //循环求和

    return 0;
}

```
