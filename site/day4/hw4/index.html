1 当执行以下程序时

x = -1;
do{
    x = x*x;
}while(!x);

循环体将会执行()

A 一次

B 两次

C 无限次

D 系统会提示语法错误

2 语句while(!e)!中的条件等价于

A e==0

B e!=1

C e!=0

D -e

3 C语言中while和do-while循环的主要区别是()

A do-while的循环体至少无条件执行一次

B while的循环控制条件比do-while的循环控制条件严格

C do-while允许从外部转到循环体内

D do-while的循环体不能是复合语句

4 以下程序段的输出结果是()

int k,n,m;
n = 10;
m = 1;
k = 1;
while(k<=n){
    m=m*2;
    k=k+4;
}
printf("%d",m);

A 4

B 16

C 8

D 32
