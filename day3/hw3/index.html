5 输出所有的水仙花数，所谓水仙花数是指一个3位数（从100到999的所有数中进行筛选），其各位数字立方和等于该数本身。例如：153是一水仙花数，因为$153 = 1^3 + 5^3 + 3^3$。

提示：一个三位数各个数位可以通过求余和除以10的方式实现。其中个位数 = 该数%10，十位数 = 该数/10 - 百位数10，百位数 = 该数/(100) 。例如：153的百位数 = 153/(100) = 1，百位数为1，十位数 = 153/10 - 110 = 5，个位数 = 153%10 = 3。

  • 输出：153 370 371 407
