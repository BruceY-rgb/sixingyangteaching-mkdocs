三、编程题（50 分）

1. 字符串连接（strcat 实现）

编写一个函数 void my_strcat(char* dest, const char* src)，实现字符串连接功能。

示例： 

char str1[20] = "Hello";
char str2[] = "World";
my_strcat(str1, str2);
printf("%s\n", str1);  // 输出 "HelloWorld"

2. 查找字符（strchr 实现）

编写一个函数 char* my_strchr(const char* str, char ch)，查找字符 ch 在 str 中第一次出现的位置，返回其指针，若未找到则返回 NULL。

示例： 

char str[] = "Hello";
char* p = my_strchr(str, 'l');
if (p != NULL) {
    printf("%s\n", p);  // 输出 "llo"
}
