作业4 答案与解析

1. 循环体执行次数

题目： 

x = -1;
do {
    x = x * x;
} while (!x);

答案：A 一次
解析：
1. 初始 x = -1。
2. 第一次循环：x = (-1) * (-1) = 1，此时 !x 为 0（假），循环终止。

2. 条件 while(!e) 的等价形式

题目：while(!e) 的条件等价于？
答案：A e==0
解析：
- !e 表示逻辑非，当 e 为 0 时 !e 为真，否则为假。
- 因此 !e 等价于 e == 0。

3. while 和 do-while 的主要区别

题目：C语言中两者的主要区别是？
答案：A do-while的循环体至少无条件执行一次
解析：
- do-while 先执行循环体，再判断条件；while 先判断条件再执行。
- 其他选项错误：
- B：条件严格性无必然关系。
- C：两者均允许 goto 跳转。
- D：两者均支持复合语句。

4. 程序段输出结果

题目： 

int k, n, m;
n = 10;
m = 1;
k = 1;
while (k <= n) {
    m = m * 2;
    k = k + 4;
}
printf("%d", m);

答案：C 8
解析：
| 循环次数 | k 值 | m 值（m *= 2） | 条件 k <= 10 |
|----------|------|------------------|----------------|
| 1 | 1 | 2 | 真 |
| 2 | 5 | 4 | 真 |
| 3 | 9 | 8 | 真 |
| 4 | 13 | - | 假（终止） |
最终 m = 8。

5. 计算前n项之和

题目：求 $1 + \frac{1}{4} + \frac{1}{7} + \cdots + \frac{1}{3n-2}$。
输入样例1：5 → 输出：1.56978
参考代码： 

#include <stdio.h>
int main() {
    int n;
    double sum = 0.0;
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        sum += 1.0 / (3 * i - 2);  // 注意用 1.0 避免整数除法
    }
    printf("%.5f\n", sum);
    return 0;
}

关键点：
- 通项公式为 $\frac{1}{3i-2}$（i 从 1 到 n）。
- 使用 1.0 确保浮点数运算。

6. 数字各位之和

题目：输入正整数，输出各位数字之和。
输入样例：123 → 输出：6
参考代码： 

#include <stdio.h>
int main() {
    int num, sum = 0;
    scanf("%d", &num);
    while (num > 0) {
        sum += num % 10;  // 取个位数
        num /= 10;        // 去掉个位数
    }
    printf("%d\n", sum);
    return 0;
}

解析：
- num % 10 获取当前个位数。
- num /= 10 去掉已处理的个位数。
